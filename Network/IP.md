# IP
```
IP = 인터넷 프로토콜의 약자.

필요한 이유: 컴퓨터는 각자의 컴퓨터마다 환경이 다르기 때문에 네트워크로 통신할 수 있도록 공통언어인 프로토콜이 필요하다.
IP 주소 : IP 통신에 필요한 고유 주소. 기기가 서로 통신을 하기 위해 필요한 고유 번호 

IPv4 : 32비트 방식이다(8비트씩 4자리 온점으로 구분)
예) 21.42.105.73 

IPv6 : 전 세계적 인터넷 사용자의 급증으로 IPv4가 고갈 위기에 처하자 이를 해결하기 위해 나온주소이다. 128비트 크기로 확장(16비트씩 8자리 콜론으로 구분.)
예) 2001:0DB8:1000:0000:0000:0000:1111:2222
```
